# DS Blog

개인 블로그 프로젝트

## 기술 스택

- **Framework** : [Astro](https://astro.build/) 5.16.6
- **Content** : MDX
- **Image Optimization** : Sharp
- **Features** :
  - RSS Feed
  - Sitemap
  - Content Collections (Blog, Project, Resume)

## 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```
