### 왜 블로그를 만들었나?

2026년이 시작되면서 새로운 것들을 시도해 보고 싶었다. 그동안 공부한 것들은 Notion을 통해 정리하고 있었는데 직접 만든 공간에 기록해 보고 싶어서 블로그를 제작했다.  
다른 사람이 봤을 때 이해가 잘되도록 정리해 보겠다.

### 사용한 기술
블로그는 Astro를 통해 제작하고 Vercel과 Gabia를 통해 배포했다.
- Astro
    - Markdown 기반 글 작성
    - 빌드 시 HTML 정적 페이지 생성
    - React 기반 프레임워크는 확장성이 뛰어나지만, 정적인 콘텐츠 중심에서는 조금 과하고 컴포넌트 구조를 신경 써야 함
    - 기존 SSG와 비교해서는 확장성이 더 낫다고 판단해 Astro 사용
- Vercel
    - GitHub 연동을 통한 자동 배포
    - 무료
    - Github Pages는 빌드 환경과 설정에 제약이 있고, 자유도가 낮다고 느껴짐
    - Netlify도 좋지만 Astro와 연계되는 측면에서 Vercel이 더 매끄러움
- Gabia
    - 개인 도메인 구매 및 DNS 관리
