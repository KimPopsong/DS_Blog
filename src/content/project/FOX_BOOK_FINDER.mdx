## [개요]
- 제16회 인천대학교 창의적종합설계경진대회
- 공공도서관이나 서점 등에서 책을 분류할 때 사용하는 한국십진분류법(KDC)은 체계적이고 간편하지만, 책이 원래의 위치에 있지 않을 경우 책을 찾을 수 없다는 단점이 있음
- 카메라로 책장을 촬영하고, 이미지 처리 및 OCR 기술을 활용하여 책의 제목을 자동으로 인식하고 위치를 파악할 수 있는 서비스
- 한국십진분류법으로 분류되지 않은 책도 쉽게 찾을 수 있도록 도와주는 시스템

## [사용 기술 스택]
- Python
    - OpenCV: 이미지 전처리
    - OCR: 책 제목 추출
- 데이터베이스: 책 정보 저장 및 관리
- 웹 크롤링: 도서관 정보 수집

## [주요 담당 기능]
- 이미지 처리
    - 카메라로 책장 사진 촬영
    - Python OpenCV를 이용한 이미지 전처리
- 책 제목 인식
    - OCR을 이용한 책 제목 추출
    - 추출된 제목을 DB에 저장
- 도서 정보 수집
    - 도서관 웹사이트 크롤링을 통한 책 정보 자동 수집
- 책 검색 기능
    - 책 제목 검색을 통해 책의 위치 파악
    - 한국십진분류법으로 분류되지 않은 책도 검색 가능
